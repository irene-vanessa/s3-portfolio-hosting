# s3-portfolio-hosting
A static portfolio website hosted on AWS S3.
This project hosts a static portfolio website using **Amazon S3** on the AWS Free Tier.

## Overview

The site showcases personal or professional projects and is built with HTML, CSS, and JavaScript. It is deployed on Amazon S3 as a static website, making it fast, scalable, and cost-effective — ideal for portfolios or landing pages.

## Key Features

- Static site hosted on **Amazon S3**
- Uses custom **HTML/CSS/JavaScript**
- Assets like images, fonts, and scripts served from S3
- Public access configured for static site hosting
- IAM setup for secure access and permissions
- Suitable for hosting resumes, projects, and portfolios

## Getting Started

To host your own version:

1. Create an S3 bucket on AWS and enable static website hosting.
2. Upload the contents of the `portfolio-folder`.
3. Set proper bucket policy to allow public access.
4. Optional: Add a custom domain or use Route 53 for DNS.
